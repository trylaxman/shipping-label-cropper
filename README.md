# Shipping Label Cropper by Baiko

A free online tool to separate Amazon and Flipkart shipping labels from invoices.

Built for Amazon, Flipkart, and marketplace sellers who frequently need shipping labels and invoices as separate PDF files.

## Features

### Amazon PDF Splitter

* Detects Amazon label PDFs
* Separates:

  * Odd pages → Shipping Labels
  * Even pages → Invoices
* Downloads two separate PDF files

### Flipkart PDF Cropper

* Detects Flipkart / Ekart shipping PDFs
* Smart dotted-line detection
* Extracts:

  * Shipping Labels
  * Tax Invoices
* Downloads separate PDF files

### Privacy First

* Files never leave your browser
* No uploads
* No registration required
* Client-side PDF processing

Install dependencies:

```bash

```

Run development server:

```bash

```

Open:

```txt
http://localhost:3000
```

Build for production:

```bash
npm run build
```
